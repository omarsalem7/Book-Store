# Book Store
This is a website to Display a list of books using an API, you can also add new book and removing it.

## Live Demo 
[Netlify demo](https://book-store-v1.netlify.app/)

## Built With

- HTML
- CSS
- ReactJS
- Redux

## Getting Started

After Cloneing

`cd "Book-Store"`
`npm install`
`npm start`
